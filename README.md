# pocket_cube_cpp

二阶魔方还原

假设转动二阶魔方任意面（或≥3阶魔方的切面）90度记为转动1步，每转动1步消耗的能量是相同的，则此程序可以称作二阶魔方的最少能量还原。

## 版本演进史

- 面向what color，即每个角块的三种颜色，存储大小`a[16][7][3]`。
- 面向where is，存储的是7个角块的位置和角块朝向，存储大小`a[16][7][2]`。
- 面向what is，存储的是7个位置分别为哪一个角块和角块朝向，存储大小`a[16][7][2]`。
